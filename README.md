# NodeJS
## myChat
一个 web 端的聊天小工具，可以通过 url 访问聊天室地址，通过网页或微
信（需要关注开发的服务号）发送信息，同时在线的用户可以收到所有人发送的信息。
同时页面会获取微信公开信息（头像和用户名等）。
